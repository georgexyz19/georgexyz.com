Vim Full Version Installation in Ubuntu

Posted on Monday, 04/01/2019 12:45 with tags: vim / ubuntu / linux

Ubuntu 18.04 comes with a stripped down version of Vim. If you want to use Vim for serious work, you need the full version.

If you type command vim in bash, it will tell you that "command vim not found". 

george@X250:~$ vim
Command 'vim' not found, but can be installed with:
sudo apt install vim
sudo apt install vim-gtk3
sudo apt install vim-tiny 
sudo apt install neovim
sudo apt install vim-athena
sudo apt install vim-gtk
sudo apt install vim-nox

Those three commands remove the existing vim-tiny and install the full version with GUI vim-gtk3 in Ubuntu 18.04 or Xubuntu. 

sudo apt-get remove vim-tiny
sudo apt-get update
sudo apt-get install vim-gtk3

You can then check the vim version by command vim --version.
